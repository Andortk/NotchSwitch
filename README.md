# MagicTap

**Tap-to-click for Apple Magic Mouse (USB-C)**

MagicTap brings tap-to-click functionality to the Apple Magic Mouse, a feature mysteriously absent from Apple's own settings.

## Features

- **Tap-to-click** - Single tap anywhere on the Magic Mouse surface triggers a left click
- **Menu bar app** - Runs quietly in your menu bar
- **Quick toggle** - Left-click the menu bar icon to enable/disable
- **Adjustable sensitivity** - Low, Medium, or High tap sensitivity settings
- **Launch at login** - Optional startup with your Mac

## Installation

1. Download the latest `MagicTap.dmg` from [Releases](../../releases)
2. Open the DMG and drag MagicTap to Applications
3. Launch MagicTap
4. Grant Accessibility permission when prompted (required for tap detection)

## Usage

- **Left-click** the menu bar icon to toggle tap-to-click on/off
- **Right-click** for settings and options
- Filled hand icon = enabled, outline = disabled

## Requirements

- macOS 13.0 (Ventura) or later
- Apple Magic Mouse (USB-C model)
- Accessibility permission

## Privacy

MagicTap requires Accessibility permission to detect touches on your Magic Mouse and simulate clicks. No data is collected or transmitted.
